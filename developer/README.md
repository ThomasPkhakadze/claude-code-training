# Claude Code Training -- Developer Track

You already know how to code. This track teaches you how to code with Claude as your copilot.

Over 4 days, you will learn Claude Code's tool system, build production dashboards at absurd speed, debug and optimize through conversation, and master the prompt patterns that make all of it possible. Everything you build follows MARTIVI DIGITAL's production standards -- the same output we ship to real clients.

---

## Track Overview

| Day | Title | What You Build | Key Claude Code Features |
|-----|-------|---------------|--------------------------|
| 1 | Power Features | One-shot dashboard from a single prompt | Tools, agentic loop, CLAUDE.md, slash commands, permissions |
| 2 | Dashboard Sprint | 6-slide client report with charts and data tables | Multi-slide HTML, Chart.js integration, CSS variables, data-driven prompts |
| 3 | Debug and Optimize | Fix a broken dashboard, make it responsive, handle client feedback | Read/Edit workflow, iterative refinement, clamp(), git branching |
| 4 | Advanced Mastery | Final test: 8-slide report + reusable template | One-shot vs iterative, prompt engineering, MCP, hooks, permission modes |

---

## Prerequisites

- Terminal proficiency (you live in the terminal)
- Git experience (branching, committing, pushing)
- HTML, CSS, and JavaScript fundamentals
- Completed the [setup guide](../setup/) -- Claude Code installed and working
- Can run `claude` in your terminal and get a response

This track does not teach you web development. It teaches you how to leverage an agentic AI tool that reads, writes, edits, and executes on your behalf.

---

## Time Commitment

**3 to 5 hours per day**, depending on how deep you go.

Day 1 and Day 4 are shorter. Day 2 and Day 3 are the heaviest. You can split any day across two sittings.
