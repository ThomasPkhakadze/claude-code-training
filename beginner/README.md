# Claude Code Training -- Beginner Track

Welcome to the Beginner Track. Over the next 7 days, you will go from someone who has never written a line of code to someone who can build professional client dashboards, visualize data with interactive charts, and deliver polished presentations -- all by having conversations with Claude Code.

You do not need any prior programming experience. You do not need to memorize anything technical. You just need to show up each day, follow the exercises, and talk to Claude Code the way you would talk to a colleague.

---

## What You Will Learn

| Day | Title | Skill You Unlock |
|-----|-------|------------------|
| 1 | First Web Page | Create web pages by talking to Claude |
| 2 | Styling | Apply professional design to any page |
| 3 | Dashboard | Build data dashboards for clients |
| 4 | Charts | Visualize data with interactive charts |
| 5 | Presentations | Create multi-slide HTML presentations |
| 6 | Prompt Engineering | Craft expert prompts that get great results |
| 7 | Git and Publish | Ship your work to GitHub and the world |

**Final Test:** Build a complete client report for Cafe Batumi from scratch.

---

## Prerequisites

Before starting Day 1, you must have completed the Setup Guide (`../setup/`). That means:

- Claude Code is installed and working on your machine
- You have a GitHub account
- You have forked or cloned the training repository
- You can open a terminal and type `claude` to start Claude Code

If any of these are not done, go back to the setup guide first.

---

## Daily Time Commitment

Each day takes **2 to 4 hours**. Some days are shorter (Day 1, Day 6), some are longer (Day 3, Day 4, Day 5). You can split a day across two sittings if needed, but try not to skip days -- each one builds on the previous.

You will be able to build professional client dashboards -- the kind agencies deliver to real clients.
