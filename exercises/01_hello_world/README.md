# Exercise 1: Hello World -- Your First Web Page

## Objective

Create a simple web page that introduces yourself. This exercise gets you comfortable with giving instructions to Claude Code and seeing the result in a browser.

## What You'll Build

A single `index.html` file with:

- A dark background
- Your name as a large heading
- A short paragraph about yourself (2-3 sentences)
- Styled text that looks clean and modern

## Instructions

1. Open Claude Code in this folder
2. Ask it to create an `index.html` file with the requirements above
3. Open the file in your browser to see the result
4. If something doesn't look right, ask Claude Code to fix it

## Example Prompt

> "Create an index.html file with a dark background (#191919), my name 'Nino' as a centered heading in white, and a short paragraph underneath that says 'I am learning to build web pages with AI.' Use a clean, modern font."

Feel free to customize the prompt with your own name and description.

## Success Criteria

- [ ] File `index.html` exists in this folder
- [ ] Page has a dark background (not white)
- [ ] Your name appears as a large heading
- [ ] There is at least one paragraph of text
- [ ] Text is readable (light text on dark background)
- [ ] Page looks centered and clean when opened in a browser

## Tips

- You can always ask Claude Code to change colors, font sizes, or spacing
- Try phrases like "make the heading bigger" or "add more spacing between elements"
- If the page looks broken, just describe what's wrong and ask for a fix
- Use Chrome or Safari to open the HTML file (right-click > Open With > browser)

## Bonus Challenge

If you finish early, try adding:
- A profile photo (use any image URL from the internet)
- A colored accent line under your name
- Links to your social media (they don't have to be real)
