# Brand Kits -- Final Project

Use the MARTIVI DIGITAL dark theme as the base for all reports. Layer client brand colors as accents on top of the standard dark theme.

---

## Base Theme: MARTIVI DIGITAL

This is the foundation for all client reports.

| Element | Value |
|---------|-------|
| Page background | #191919 |
| Card background | #1e1e1e |
| Card border | #2a2a2a |
| Primary text | #ffffff |
| Secondary text | #e0e0e0 |
| Muted text | #888888 |
| Primary accent | #ffc711 (Yellow) |
| Secondary accent | #3cc5ee (Cyan) |
| Heading font | Montserrat (Google Fonts) |
| Body font | Inter (Google Fonts) |

## Client: Cafe Batumi

A warm, coffee-inspired palette. Use these colors for headings, accents, chart elements, and highlights. The dark MARTIVI base theme stays as-is for backgrounds and cards.

| Element | Value | Usage |
|---------|-------|-------|
| Primary | Coffee Brown #6F4E37 | Section borders, table headers, chart bars |
| Secondary | Cream #FFFDD0 | Accent text, highlighted numbers, card labels |
| Accent | Espresso #3C1F0A | Hover states, secondary borders |
| Chart color 1 | #6F4E37 | Primary data series |
| Chart color 2 | #FFFDD0 | Secondary data series |
| Chart color 3 | #A0785A | Tertiary data series (lighter brown) |

**Suggested approach:** Replace the MARTIVI yellow (#ffc711) accent with Coffee Brown (#6F4E37) for primary highlights, and use Cream (#FFFDD0) where you would normally use cyan (#3cc5ee). Keep the dark backgrounds unchanged.

---

## Client: TechFlow

A sharp, tech-forward palette. Blues and navys convey professionalism and trust -- appropriate for a SaaS board presentation.

| Element | Value | Usage |
|---------|-------|-------|
| Primary | Electric Blue #0066FF | Section headings, primary chart color, CTA buttons |
| Secondary | Dark Navy #0A1628 | Can replace #1e1e1e for card backgrounds for deeper feel |
| Accent | Light Gray #F0F4F8 | Stat card values, highlighted numbers |
| Chart color 1 | #0066FF | Primary data series |
| Chart color 2 | #00AAFF | Secondary data series (lighter blue) |
| Chart color 3 | #FF6B35 | Accent/contrast data series (warm orange) |
| Chart color 4 | #10B981 | Positive indicators (growth, success) |
| Chart color 5 | #EF4444 | Negative indicators (decline, churn) |

**Suggested approach:** Replace the MARTIVI yellow (#ffc711) accent with Electric Blue (#0066FF) for primary highlights. Use #00AAFF for secondary elements. Keep stat card numbers white or Light Gray (#F0F4F8) for readability. The Dark Navy (#0A1628) can be used for card backgrounds instead of the standard #1e1e1e for a more immersive tech feel.

---

## Notes for Both

- Always ensure sufficient contrast between text and backgrounds (WCAG AA minimum)
- The MARTIVI DIGITAL logo/brand name in the footer should always use the standard yellow #ffc711 regardless of client branding
- Charts should use the client palette, not the MARTIVI palette
- Test the final result in a browser to make sure colors look right on screen -- some colors that look good in a hex picker can feel washed out on a dark background
