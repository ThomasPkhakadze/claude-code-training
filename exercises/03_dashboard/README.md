# Exercise 3: Social Media Analytics Dashboard

## Objective

Build a complete analytics dashboard using real-looking data. This is where things get interesting -- you'll work with a JSON data file and create a page with multiple visualization types.

## What You'll Build

A single-page HTML dashboard for **Cafe Batumi** showing their March 2026 social media performance. The dashboard must include:

1. **Header** -- Client name, report period, and MARTIVI DIGITAL branding
2. **Stat Cards** -- Key metrics displayed as large numbers with labels (Reach, Engagements, Total Spend, etc.)
3. **Data Table** -- A table listing individual ad campaigns with their performance metrics
4. **Bar Chart** -- A visual chart showing monthly trends (use Chart.js library)
5. **Platform Breakdown** -- Separate sections or visual distinction for Meta vs Google data

## Data Source

Use the `data.json` file in this folder. It contains all the metrics you need for Cafe Batumi.

Read the data file first to understand the structure, then build the dashboard around it.

## Design Specifications

| Element | Value |
|---------|-------|
| Background | #191919 |
| Card background | #1e1e1e or #2a2a2a |
| Primary accent | #ffc711 (MARTIVI yellow) |
| Secondary accent | #3cc5ee (MARTIVI cyan) |
| Font | Montserrat (Google Fonts) |
| Chart library | Chart.js (CDN) |
| Max width | 1200px, centered |

## Instructions

1. Start by reading `data.json` to understand what data is available
2. Ask Claude Code to build a dashboard that presents this data
3. Be specific about what you want: "Show the Meta ads in a table with columns for name, reach, clicks, CTR, and spend"
4. Iterate on the design until it looks like a professional agency report

## Success Criteria

- [ ] Dashboard displays in a single HTML file
- [ ] All data comes from `data.json` (numbers must match the file)
- [ ] At least 4 stat cards showing key metrics
- [ ] A data table with all 5 Meta ad campaigns
- [ ] A bar chart showing the monthly trend data (6 months)
- [ ] Google Ads data is displayed (table or cards)
- [ ] Dark theme with MARTIVI branding colors
- [ ] Montserrat font loaded and applied
- [ ] Page looks professional enough to send to a client

## Tips

- You can embed the JSON data directly in the HTML file using a `<script>` tag, or you can hardcode the values -- either approach works
- For Chart.js, include this CDN link: `https://cdn.jsdelivr.net/npm/chart.js`
- Ask for specific chart types: "Make a bar chart with reach on the left axis and spend on the right axis"
- If the chart doesn't render, check the browser console (right-click > Inspect > Console tab) for errors
- Tables look better with alternating row colors and proper alignment

## Bonus Challenge

- Add a second chart type (line chart, doughnut chart, etc.)
- Add a "Platform Comparison" section showing Meta vs Google side by side
- Make the stat cards show an arrow icon (up/down) with a percentage change
- Add a footer with "Report generated by MARTIVI DIGITAL"
- Export-friendly: make it look good if someone prints the page (add `@media print` styles)
