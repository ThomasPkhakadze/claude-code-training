# Claude Code Training Program

> **By MARTIVI DIGITAL** -- Learn to build professional HTML dashboards and landing pages using Claude Code.

---

## What Is This?

This is a hands-on training program that teaches you to use **Claude Code** -- an AI-powered development tool that runs in your terminal. By the end, you'll be able to build professional client-ready dashboards, presentations, and landing pages.

Every day you'll build something real. By Day 3, your work will look like what we deliver to actual clients.

## How This Training Works

### Daily Structure

Each day follows this pattern:

1. **Read** the day's lesson (10-15 min)
2. **Do** the exercises with Claude Code (1-3 hours)
3. **Commit** your work to GitHub
4. **Update** your progress checklist

### What You'll Build

| Exercise | What You Create |
|----------|----------------|
| 01 | A personal greeting web page |
| 02 | A styled profile page with stat cards |
| 03 | A social media analytics dashboard |
| 04 | A multi-slide HTML presentation |
| 05 | Debug a broken dashboard (developer only) |
| 06 | **Final project:** Complete client report from a brief |

### Progress Tracking

Your progress is tracked through GitHub commits:

- After each exercise, commit your work with the format: `day-X: brief description`
- Push to your fork so your progress is visible
- Your supervisor can review your work anytime via GitHub

## The Agency Standard

Everything you build in this training follows MARTIVI DIGITAL's production standards:

- **Dark theme** -- Background `#191919`, cards `#262626`
- **Brand colors** -- Yellow `#ffc711`, Cyan `#3cc5ee`
- **Font** -- Montserrat (Google Fonts)
- **Charts** -- Chart.js via CDN
- **Self-contained** -- Single HTML files, no build tools
- **Professional** -- Footer with MARTIVI DIGITAL branding

By the final test, your output will match what we ship to real clients.
