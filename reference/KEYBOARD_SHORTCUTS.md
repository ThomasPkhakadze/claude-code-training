# Keyboard Shortcuts

## Inside Claude Code

| Action | Mac | Windows/PC |
|--------|-----|------------|
| **Send message** | `Enter` | `Enter` |
| **New line** (without sending) | `Option + Enter` | `Shift + Enter` (some terminals) or `Ctrl + J` |
| **Cancel generation** | `Escape` | `Escape` |
| **Exit Claude Code** | `Ctrl + C` or `Ctrl + D` | `Ctrl + C` or `Ctrl + D` |
| **Recall previous message** | `Up Arrow` | `Up Arrow` |
| **Clear screen** | `Ctrl + L` | `Ctrl + L` |
| **Run shell command** | `! command` | `! command` |
| **Cycle permission mode** | `Shift + Tab` | `Shift + Tab` |
| **Switch model** | `Option + P` | `Alt + P` |
| **Toggle fast mode** | `Option + O` | `Alt + O` |
| **Toggle thinking mode** | `Option + T` | `Alt + T` |

## In Terminal (Outside Claude Code)

| Action | Mac (Terminal) | Windows (PowerShell) |
|--------|---------------|---------------------|
| **Open terminal** | `Cmd + Space` → type "Terminal" | `Windows key` → type "PowerShell" |
| **Copy** | `Cmd + C` | `Ctrl + C` (when text selected) |
| **Paste** | `Cmd + V` | `Ctrl + V` or right-click |
| **Cancel running command** | `Ctrl + C` | `Ctrl + C` |
| **Clear screen** | `Cmd + K` or `clear` | `cls` |
| **Previous command** | `Up Arrow` | `Up Arrow` |
| **Open file in browser** | `open filename.html` | `start filename.html` |
| **Tab completion** | `Tab` | `Tab` |

## When Claude Asks for Permission

When you see `Allow? [y/n]`:

| Action | Key |
|--------|-----|
| **Accept** (yes, do it) | `y` then `Enter` |
| **Reject** (no, don't) | `n` then `Enter` |
| **Always allow this tool** | `a` then `Enter` |

## Pro Tips

- **`!` is your friend** -- Run any shell command without leaving Claude. Example: `! open exercises/03_dashboard/dashboard.html` to preview your work.
- **`Up Arrow` saves time** -- Recall and edit your previous message instead of retyping.
- **`Escape` is your safety net** -- If Claude is generating something wrong, hit Escape to stop it immediately.

---

*Tip: These shortcuts work in most terminals. If a shortcut doesn't work, your terminal app may use different bindings.*
